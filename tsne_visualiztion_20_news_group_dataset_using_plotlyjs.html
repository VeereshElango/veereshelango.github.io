The aim of this visualization is to understand the working of t-SNE.

Approach

This scatter plot is generated by parsing the text of each article. Then each instance is turned into features vector using Sklearn's TfidfVectorizer. Then, dimensionality reduction is applied to the feature vectors using Sklearn's t-SNE. Total of 5627 articles from 10 different target groups are analysed.
